개행 문자 포맷팅 도구

- 입력 문자열의 개행 문자를 제거하거나, 반대로 텍스트인 개행 문자를 실제 줄바꿈으로 바꾸거나, 선택한 옵션에 따라 문자열을 수정하여 표시합니다.

- 포맷팅 옵션을 선택하고, 입력창에 문자열을 입력한 후 '포맷팅' 버튼을 누르면 결과가 출력됩니다.

포맷팅 옵션

입력

임시 저장되었습니다.

공백 처리 옵션

결과

사용 예시

1. 텍스트 개행 문자를 실제 줄바꿈으로 변환
- 입력 문자열
Hello, World!\r\n안녕하세요.\r\n안녕하세요.
- 포맷팅 결과
Hello, World!
안녕하세요.
안녕하세요.
2. 개행 문자 제거
- 입력 문자열
Hello, World!\r\n안녕하세요.\r\n안녕하세요.
- 포맷팅 결과
Hello, World!안녕하세요.안녕하세요.
3. 개행 문자를 <br> 태그로 변환
- 입력 문자열
Hello, World!\r\n안녕하세요.\r\n안녕하세요.
- 포맷팅 결과
Hello, World!
안녕하세요.
안녕하세요.
4. 개행 문자를 사용자 지정 태그로 감싸기
- 입력 문자열
Hello, World!\r\n안녕하세요.\r\n안녕하세요.
- 포맷팅 결과
<div>Hello, World!</div><div>안녕하세요.</div><div>안녕하세요.</div>
5. 개행 문자를 입력한 문자열로 변환
- 입력 문자열
Hello, World!\r\n안녕하세요.\r\n안녕하세요.
- 포맷팅 결과
Hello, World! [사용자 지정 문자열] 안녕하세요. [사용자 지정 문자열] 안녕하세요.
